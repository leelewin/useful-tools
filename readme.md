# useful-tools
## 1.docker
docker_run.sh

useful docker shell script

#### script1：docker_run.sh
这个脚本用来保存docker容器到新的镜像名，然后将新的镜像名改为原来的，视觉上就是在源镜像上进行了修改    
包含很多交互式提示很方便。

用法：
`bash docker_run.sh`
